# Sales Methodology Enrichment Summary

## What Was Added

This enriched version of `sales-methodology` skill adds comprehensive SPICED qualification depth to the original framework.

### New Sections

1. **SPICED Deep Dive: Element-by-Element**
   - Full depth on each SPICED component (Situation, Pain, Impact, Critical Event, Decision)
   - Scoring rubric for each element with examples
   - What to uncover in discovery for each element
   - Red flags and warning signs

2. **SPICED Scoring in Practice: Real Deal Example**
   - End-to-end worked example showing how to score a €80K deal
   - Evidence mapping for each SPICED element
   - Coaching debrief showing how to identify gaps and next actions

3. **Buying Committee Mapping: SPICED Language by Persona**
   - SPICED language customised for each stakeholder role:
     - CRO (Executive Buyer)
     - VP Sales (Champion, User Buyer)
     - Sales Operations / Enablement (Technical Buyer)
     - CFO / Finance (Fiscal Gatekeeper)
     - CTO / Head of IT (Technical Buyer)
   - Role-specific objection handling and key questions

4. **Discovery Call Structure Using SPICED**
   - 45-minute discovery call template structured around SPICED elements
   - Pre-call prep checklist
   - Post-call scoring and action planning
   - Example discovery call transcript with live scoring

5. **SPICED Qualification Scoring by Customer Cluster**
   - Quick reference table for three customer clusters (PE, Large SaaS, Small SaaS)
   - Different SPICED language patterns by cluster
   - How to use clusters for deal routing and discovery tailoring

6. **SPICED Evidence by Pipeline Stage**
   - Minimum SPICED scores required at each stage (Discovery, Qualification, Solution Design, Negotiation)
   - Red flags to exit or delay deals
   - What evidence must be captured before stage advancement

### Reference Files Added

- `references/spiced-icp-library.md` — Complete SPICED language library with canonical phrases for each customer cluster (PE, Large SaaS, Small SaaS), qualified tiers (T1/T2/T3), and expanded SPICED elements
